# Basic Widgets and Layouts in Flitter

In this tutorial, we'll explore some of the basic widgets in Flitter and learn how to combine them to create simple layouts. We'll focus on `Container`, `Row`, `Column`, and `Text` widgets, and use them to build a basic card layout.

## Key Widgets

1. **Container**: A convenience widget that combines common painting, positioning, and sizing widgets.
2. **Row**: A widget that displays its children in a horizontal array.
3. **Column**: A widget that displays its children in a vertical array.
4. **Text**: A widget for displaying text with a single style.
5. **EdgeInsets**: A utility class for describing offsets in terms of visual edges.

## Building a Basic Card Layout

The example code demonstrates how to create a simple card layout using these widgets. Let's break it down:

1. We start with a `Container` as the root, giving it a background color and padding.
2. Inside this, we use a `Column` to arrange elements vertically.
3. The first child of the column is a `Text` widget for the title.
4. We then use another `Container` as a spacer.
5. The main content is a `Row`, containing an image placeholder and some text.

## Understanding the Layout

- The `Row` widget arranges its children horizontally.
- We use `Container` with specific dimensions for the image placeholder.
- The `Expanded` widget is used to make the text take up all remaining horizontal space.
- We use nested `Column` widgets to arrange text vertically within the `Row`.

## Key Concepts

- **Nesting**: Flitter layouts often involve nesting widgets inside each other to create complex structures.
- **Flexibility**: Widgets like `Expanded` allow for flexible layouts that adapt to different screen sizes.
- **Spacing**: Use `Container` with specific dimensions as spacers between elements.
