# Introduction to flitter-react

flitter-react is a React wrapper for the Flitter library, providing an easier way to integrate Flitter widgets into React applications. It simplifies the process of rendering Flitter widgets by handling the AppRunner setup and lifecycle management for you.

## Getting Started

To use flitter-react, you need to install both flitter and flitter-react:

```bash
npm install @meursyphus/flitter @meursyphus/flitter-react
```

## Basic Usage

The example code demonstrates how to use flitter-react to render a simple Flitter widget. Here's what's happening:

1. We import the `Widget` component from flitter-react.
2. We create a Flitter widget using `Container` and `Text` components from flitter.
3. We use the `Widget` component to render our Flitter widget, specifying the width and height.

## Advantages of flitter-react

- **Simplified Integration**: No need to manually manage AppRunner or SVG references.
- **React-like Usage**: Feels more natural in a React application.
- **Automatic Lifecycle Management**: The Widget component handles mounting, unmounting, and resizing for you.

## Key Concepts

- **Widget Component**: The main component from flitter-react used to render Flitter widgets.
- **Flitter Components**: We still use Flitter components (Container, Text, etc.) to build our widget tree.
