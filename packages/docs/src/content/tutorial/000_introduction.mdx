      const widget = Container({
        width: Infinity,
        height: Infinity,
        alignment: Alignment.center,
        color: "lightblue",
        child: Text("Welcome to Flitter!")
      });
      
      useEffect(() => {
        runner = new AppRunner({
          view: svgRef.current
        });
        runner.onMount({
          resizeTarget: svgRef.current
        });
        runner.runApp(widget);
        return () => {
          runner.dispose();
        };

# Introduction to Flitter

Flitter is a powerful library for creating interactive data visualizations in React applications. It provides a declarative way to build complex visualizations without direct DOM manipulation.

## Getting Started

To use Flitter in your React project, first install it:

```bash
npm install @meursyphus/flitter
```

Then, you can start using Flitter components in your React components.

## Basic Usage

The example code demonstrates how to render a simple Flitter widget using the `AppRunner` class. Here's a breakdown of what's happening:

1. We import necessary components from Flitter.
2. We create a widget using `Container` and `Text` components.
3. We use `useRef` to get a reference to the SVG element.
4. In the `useEffect` hook, we create an `AppRunner` instance, set up resizing, and run our widget.
5. We render an SVG element that Flitter will use to draw our widget.

## Challenge

Try modifying the widget to display two lines of text instead of one. Use a `Column` widget to stack the text vertically. You can use the code editor to make changes and see the results in real-time.

When you're ready, click the "Solve" button to see one possible solution.

## Key Concepts

- **AppRunner**: This class is responsible for managing the Flitter application lifecycle.
- **Container**: A basic layout widget that can contain other widgets.
- **Text**: A widget for displaying text.
- **Alignment**: Used to position widgets within their parent containers.

In the next tutorial, we'll explore more widgets and learn how to create more complex layouts.
